  • Enhanced Artifact: binarySearch.js, binarySearch.test.js
The goal of this enhancement was to demonstrate algorithm optimization by replacing linear search with a binary search implementation. I created a utility function for efficient lookups by ID and integrated it into both monkey and dog search routes. The enhancement also included unit tests to validate correctness. This shows how I applied algorithmic principles to improve performance and reduce time complexity from O(n) to O(log n), demonstrating real-world problem solving in data handling.
