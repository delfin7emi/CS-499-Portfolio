üîß Original Artifacts

For the CS-499 Capstone project, students are asked to select one or more artifacts in the following categories:

  • Software Engineering and Design
  • Algorithms and Data Structures
  • Databases

Artifacts may come from prior SNHU courses, projects from work, or open-source projects. As a demonstration, I have selected the following artifacts:

  • Software Engineering and Design: A full-stack rescue animal manager originally written in Java.
  • Algorithms and Data Structures: The same application extended with custom data structures and searching techniques.
  • Databases: The Grazioso Salvare Pet Project from IT-145 extended with MongoDB and Mongoose.
